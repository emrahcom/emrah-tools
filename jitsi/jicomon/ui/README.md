# Jicomon UI

#### installation

- Install `nodejs` and `git` packages.

- Get a copy of the repo (don't run as `root`):

- Install `npm` modules (don't run as `root`):

```bash
cd ui
npm install
```

#### running (dev)

```bash
npm run dev -- --host --port 3000
```

#### running (prod)

```bash
npm run build
node build/index.js
```
